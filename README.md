# AlterForm

AlterForm is a starter 2D wardrobe constructor. It lets you choose a mannequin, add visual layers, save a look in the browser, and export the result as PNG.

## Features

- 3 mannequin bases
- Hair, makeup, dresses, tops, bottoms, shoes, and accessories
- Layered SVG assets
- Clear current look
- Save looks with localStorage
- Export current look to PNG
- Responsive layout

## GitHub Pages

Open Settings -> Pages and use:

- Source: Deploy from a branch
- Branch: main
- Folder: /root

Then press Save.

## Adding items

Add a new asset to assets/clothes and register it in app.js inside the clothes array.
